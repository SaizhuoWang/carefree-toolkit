# carefree-toolkit

`carefree-toolkit` implemented some commonly used functions and modules


## Installation

`carefree-toolkit` requires Python 3.6 or higher.

```bash
git clone https://github.com/carefree0910/carefree-toolkit.git
cd carefree-toolkit
pip install -e .
```
